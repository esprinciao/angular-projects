Types of tags

Container tags which dont have their own content eg.table, body, head, div

Standalone tags which need not be closed eg. br, image, hr

Content tags which must be open and closed, the content is entered b/w opening and closing tags eg.h1,td,tr

Parent and child tags eg. table,tr,td

Tags have attributes - like id, class, style
